## Стратегия

**ORB (Opening Range Breakout)** — Toby Crabel, 1990.

- Диапазон первых 2 свечей после открытия сессии
- Вход на пробой диапазона
- Стоп за противоположную границу
- Тейк 1:2 к стопу
- Winrate ~38%, матожидание +0.14 на рубль риска (7000 сделок, 1999–2025)

## Инструменты

- **Исполнение:** Tinkoff Invest API (Python SDK)
- **Бэктест:** исторические данные MOEX
