# Simple Strategy — ORB Trading Robot

## Цель проекта

Разработать торгового робота на Python для российского рынка (MOEX) на основе стратегии ORB (Opening Range Breakout). Брокер — Tinkoff Invest API.

## Стратегия ORB — ключевые параметры

- **Диапазон:** первые 2 свечи M5 после открытия основной сессии MOEX (10:00 МСК)
- **Вход:** пробой верхней/нижней границы диапазона
- **Стоп:** за противоположную границу диапазона
- **Тейк:** 1:2 к стопу
- **Фильтр:** не входить против VWAP
- **Лимит:** одна сделка в день на инструмент
- **Статистика:** winrate ~38%, матожидание +0.14 на рубль риска, 7000 сделок (1999–2025)

## Инструменты для торговли

- **Si** (фьючерс USD/RUB) — основной
- **Br** (фьючерс нефть Brent) — альтернативный
- Можно расширить на акции MOEX

## Технический стек

- **Python 3.11+**
- **tinkoff-invest-python** — получение данных и исполнение ордеров
- **pandas** — работа с данными
- **Бэктест:** исторические свечи через Tinkoff API
